What's this site about?

This site is designed to provide a visual intuition for what a few common machine learning models do and how their parameters influence that behavior. All the models are implemented from scratch using numpy with the guiding principle to keep the implementation at the level of linear algebra, or in other words, no for loops allowed! Check out the implementation on Github.

Check out the available plots in the navigation bar. Click on any plot to add a point and watch the model update accordingly. Some plots have sliders to adjust parameters, which will also update the model.
